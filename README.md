# SlicePress — 3D Print Quoting for WooCommerce

SlicePress adds a fully interactive 3D model viewer and instant print quoting to any WooCommerce store. Customers upload STL or 3MF files, preview them in-browser, choose material and quality, get an instant price quote, and add directly to the cart.

**Features:** Three.js 3D viewer · Smart auto-orient (OrcaSlicer algorithm) · STL + 3MF support · Instant quoting · Theme-adaptive design · HPOS compatible · Admin 3MF download per order

Requires a [SlicePress API key](https://slicepress.druckhochdrei.com). The backend and slicer service are proprietary — this plugin is the open-source client.
